This project is a Python-based AI Agent that helps job seekers by:

- Reading a Job Description (JD) from PDF, DOCX, or TXT
- Extracting structured requirements (title, skills, responsibilities, must-have and nice-to-have requirements)
- Generating a tailored cover letter automatically
- Producing a 1-minute video script introducing the candidate

## Features
- Upload JD and resume or paste profile text
- Automatically extract key requirements
- Generate a professional, tailored cover letter
- Create a short video script for self-introduction
- Supports multiple cover letter tones: professional, friendly, direct, enthusiastic
