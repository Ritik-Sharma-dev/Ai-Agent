# AI Agent — JD to Cover Letter + Video Script

This project is a Python-based AI Agent that:
- Reads a Job Description (JD)
- Extracts structured requirements
- Drafts a tailored cover letter
- Generates a 1-minute video script introducing the candidate

## Features
- Upload JD and resume (PDF, DOCX, TXT)
- Extract key requirements automatically
- Generate a professional cover letter
- Generate a short video script for self-introduction
